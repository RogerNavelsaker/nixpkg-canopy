# Canopy

Use this skill when the task is about creating, organizing, reviewing, or applying prompt assets managed by Canopy.

Preferred commands:

```bash
canopy --help
canopy list
canopy status
```

Rules:
- Use canopy for reusable prompts and prompt organization.
- Keep prompts versioned and inspectable in git.
- Prefer editing the source prompt asset instead of copying prompt text around.
